ψ(xA, xB) = ψ1(xA)ψ2(xB) + ψ1(xA)ψ3(xB) + ... + ψn(xA)ψm(xB)

For example, if two particles are entangled in such a way that their spins are correlated, the correlated wave function would reflect this correlation. The wave function would be zero if the spins are not correlated, and would have a non-zero value if the spins are correlated.
