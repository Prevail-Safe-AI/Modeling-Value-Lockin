Examples of structured data include:

* CSV (Comma Separated Values) files, where each row represents a single data point, and each column represents a specific attribute or feature.
* JSON (JavaScript Object Notation) objects, which are used to store and exchange data in a human-readable format.
* Relational databases, such as MySQL or PostgreSQL, which store data in tables with defined schemas.
