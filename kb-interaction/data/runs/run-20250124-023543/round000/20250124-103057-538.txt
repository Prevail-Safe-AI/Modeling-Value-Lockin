Template-based response generation is a technique used in conversational AI dialog generation to generate responses to user inputs. In this approach, the AI system uses pre-defined templates or structures to generate responses. These templates can be based on various factors such as:

1. User input: The AI system can analyze the user's input and use it to fill in the blanks in a pre-defined template.
2. Context: The AI system can use the context of the conversation to determine which template to use.
3. Intent: The AI system can analyze the user's intent and use it to select a relevant template.

Templates can be in the form of:

1. Sentence templates: These are pre-defined sentence structures that can be filled in with relevant information.
2. Slot-filling templates: These are templates that have placeholders for specific information, such as names, dates, or locations.
3. Frame-based templates: These are templates that represent a specific concept or scenario, such as a booking or a reservation.

The process of template-based response generation typically involves the following steps:

1. Text analysis: The AI system analyzes the user's input and determines the relevant information.
2. Template selection: The AI system selects a relevant template based on the user's input and context.
3. Filling in the blanks: The AI system fills in the placeholders in the selected template with the relevant information.
4. Response generation: The AI system generates a response by combining the filled-in template with additional information, such as context and intent.

For example, if a user asks "What is the weather like in New York today?", the AI system might use a template-based response generation approach to generate a response like "The weather in New York today is sunny with a high of 75°F."
